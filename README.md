Bot Discord przeznaczony dla serwerów FiveM. Umożliwia sprawdzanie statusu serwera, liczbę graczy oraz posiada podstawowe komendy administracyjne, takie jak whitelist.
### Status serwera
-  Sprawdza, czy serwer FiveM jest online.
-  Wyświetla podstawowe informacje o serwerze.
### Liczba graczy (w statusie bota)
- Komenda do sprawdzenia aktualnej liczby graczy.
- Możliwość wyświetlania listy graczy (jeśli API serwera na to pozwala).
### Whitelist
- Dodawanie i usuwanie graczy z whitelisty.
- Obsługa podstawowych funkcji administracyjnych.
### Podstawowe komendy
- Komenda pomocy /help
- Logi Pod Webhook
- Komendy moderacyjne (opcjonalnie)
- Możliwość rozszerzania o kolejne funkcje 
### Instalacja przebiega bardzo prosto:
1. Pobierz plik .zip z botem.
2. Wypakuj folder z pliku .zip.
3. Wgraj wypakowany folder do głównego katalogu serwera FiveM, np. `resources/[bot]`
4. W pliku `server.cfg` dodaj `ensure [bot]`
5. Uruchom serwer.

### Download Link [scode_bot.zip](https://github.com/user-attachments/files/23940790/scode_bot.zip)
Discord bot designed for FiveM servers. It allows you to check the server status, number of players, and has basic administrative commands such as whitelist.
### Server Status
- Checks if the FiveM server is online.
- Displays basic information about the server.
### Number of Players (in bot status)
- Command to check the current number of players.
- Ability to display a list of players (if the server API allows it).
### Whitelist
- Adding and removing players from the whitelist.
- Support for basic administrative functions.
### Basic commands
- Help command /help
- Webhook logs
- Moderator commands (optional)
- Possibility to extend with additional functions 
### Installation is very simple:
1. Download the .zip file with the bot.
2. Extract the folder from the .zip file.
3. Upload the extracted folder to the main directory of the FiveM server, e.g., `resources/[bot]`
4. In the `server.cfg` file, add `ensure [bot]`
5. Start the server.
